
## STOPWATCH

A simple stopwatch application built with React.It demonstrate how to manage tasks, handle events.This app let you start, stop, and reset the timer.

---

## Features

1. Start/Stop buttton to on and off
2. Reset button to clear the time back to zero
3. Clean UI for more focus on functionality
4. Show time in hours, mins, sec, millisec

---

## Technologies

- **React**   &nbsp;->Component, hooks(useState, useEffect)
- **CSS**    &nbsp; -> style for some button and layout
